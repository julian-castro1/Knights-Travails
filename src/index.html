Knight's Travails
Instructions: Click a spot on the chess board to place the knight
Result: -

How it works:

This is an implementation of the 'Knight's Travails' problem using search algorithms. The goal of this program is to determine the shortest (least amount of moves) way to get from the starting point to the end point using a knight's moving pattern. You can follow the instructions on the left, or read below. To the left is an 8x8 chess board. To begin, select a starting point for the knight and click 'Next'. Next, choose the ending location and click 'Solve'. The program will now use a search algorithm to determine the shortest path to the selected location and display the path.
